الخبرة المهنية (Industry‑Ready)

تحليلات تشغيلية وبيانات حسّاسات

  • قراءة بيانات SCADA/Telemetry وربطها بالسياق التشغيلي (ضغط/تدفق/حرارة…)
  • Dashboards لمؤشرات الأداء (KPIs) + تقارير مقارنة + تتبع اتجاهات الأداء.
  • كشف الشذوذ (Anomaly Detection) على السلاسل الزمنية + إنذارات (alarms) لتقليل التوقفات.
  • نماذج تنبؤية للصيانة (Predictive Maintenance) ودعم القرار التشغيلي.
  • التزام بإجراءات السلامة HSE والعمل ضمن بيئات تشغيلية (مناوبات/Handovers).

هندسة بيانات + تطوير

  • ETL وتنظيف بيانات + فحص جودة + توحيد مخططات (Schema).
  • تحليلات نصية عربية (NLP) + Fine‑tuning + تقييم (Precision/Recall/F1).
  • كتابة تقارير يومية/أسبوعية باللغة الإنجليزية عند الحاجة، وتوثيق إجراءات التشغيل.
